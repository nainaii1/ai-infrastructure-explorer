# ⚡ AI Infrastructure Explorer

An interactive, offline-first map of the **AI hardware supply chain** — how NVIDIA + the hyperscalers connect to the layers of suppliers beneath them (photonics, memory, fabs, neoclouds, materials, networking). Click a value-chain layer to filter the companies; track them in a watchlist with weekly prices; and feed in sourced theses from an analyst you follow.

> Personal research tool. Single-user, local-first. Not investment advice.

## What it does
- **Supply Chain Map (Tab 1)** — a bottom→top *value-chain layer* stack (materials → memory → photonics → fabs → networking → neoclouds → demand hub). Click a layer to filter the company cards below; each card shows what the company does, why NVDA needs them, market, and cap tier.
- **Watchlist (Tab 2)** — a sortable table of every tracked ticker: category, price, **7D / 1M %**, market cap, and your own rating. Prices refresh **weekly** from Yahoo via a one-click button (server-side; see below).
- **Thesis (Tab 3)** — a feed of ideas ingested from **@aleabitoreddit**'s posts, ranked by a composite **priority** score (mention frequency + recency + conviction language). *(Tab UI in progress; the ingest pipeline is built.)*

## Run it
**Just open the app** — double-click `index.html`. It works fully offline over `file://` (no server, no build, no network needed). All data lives in `data.js`.

**For live prices** (the Watchlist "Fetch prices" button needs a local helper, because browsers can't call Yahoo from `file://`):
```bash
python3 ingest/serve.py        # then open http://localhost:8765/
```
Go to **Watchlist → Fetch prices**. It pulls Yahoo server-side, computes 7D/1M %, and rewrites `data.js`. Plain double-click still works afterward, showing the last-fetched prices.

## Backend (`ingest/`)
Local Python tooling (standard library only — nothing to `pip install`) that **regenerates `data.js`**. It never ships to the browser.
- **Telegram ingest** — forward an @aleabitoreddit post to your bot; it becomes a *thesis* (linked to the source), auto-adds clean `$CASHTAG` tickers (queues ambiguous ones), and updates the priority ranking.
- **Price fetch** — `fetch_prices.py` / `serve.py` bake weekly Yahoo prices into `data.js`.
- See [docs/TELEGRAM_SETUP.md](docs/TELEGRAM_SETUP.md) for setup (BotFather token in `ingest/.env`, etc.).

## Tech & constraints
- **Vanilla HTML / CSS / JS** — no frameworks, no CDN, no build step.
- **Offline-first**: must work by double-clicking `index.html`; `data.js` is loaded via `<script>` (no `fetch()` of local files).
- `data.js` is the **single source of truth**, regenerated by the Python tooling.
- Secrets live in a gitignored `ingest/.env` (never committed).
