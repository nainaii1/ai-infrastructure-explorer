05 / Guardrails

Consistency is the feature.

These are implementation constraints as much as visual preferences.

Offline by default

Keep pages vanilla and file://-safe. Load data.js before page logic; do not fetch local data in-browser.

Data owns category colour

Read category colour, labels, and ticker information from AIE_DATA. Do not hardcode a category palette into page CSS or scripts.

Motion earns attention

Use 240ms motion for hover and state changes; reserve the spring curve for pills and measured drill-downs. Respect reduced-motion preferences.

Responsive at 768px

Mobile is a reordered reading experience: tables stack, navigation wraps, and dense grids simplify. Do not merely shrink the desktop view.
